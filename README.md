# Информация о храме Трех Святителей в Голенищеве

Храм Трех Святителей: Василия Великого, Григория Богослова и Иоанна Златоуста - был основан в XIV веке святителем Киприаном, митрополитом Московским, в селе Голенищеве, которое в то время располагалось близ Москвы, в месте впадения реки Раменки в Сетунь.

Настоящая страница создана с целью сбора всей доступной информации об истории храма Трех Святителей. Для этого мы собираем:
 * Источники (летописи, архивные документы)
 * Научные статьи
 * Научно-популярные статьи (в том числе интернет-публикации)
 
Надеемся, что данная деятельность позволить упорядочить разрозненные факты, статьи, разбросанные в разных книгах, журналах, интернет-ресурсах и т.д.
